Криптология, Криптография, Криптоанализ
Криптология - наука о создании и анализе систем безопасной связи.
Криптография - наука о методах обеспечения секретности и/или подлинности (идентичности) данных при их передаче по линиям связи или хранения.
Криптоанализ - наука о методах раскрытия и/или подделки данных.
Криптография и криптоанализ нацелены на решение взаимно обратных задач.
Шифрование (криптографическое закрытие информации) заключается в таком преобразовании защищаемой информации, при котором по внешнему виду нельзя определить содержание закрытых данных.
Замена
Шифрование заменой (подстановкой) заключается в том, что символы шифруемого текста заменяются символами того же или другого алфавита в соответствии с заранее обусловленной схемой замены.
В зависимости от числа используемых алфавитов различают следующие схемы замены:
Моноалфавитная замена
Многоалфавитная замена
G-контурная многоалфавитная замена
Моноалфавитная замена
Методы моноалфавитной замены:
- Шифр Цезаря;
- Прямая моноалфавитная подстановка (замена).
Все методы моноалфавитной замены можно представить как числовые преобразования букв исходного текста, рассматриваемых как числа.
Прямая моноалфавитная подстановка (замена)
Каждая буква в тексте (Р) умножается на некоторое число (называемое десятичным коэффициентом) и прибавляется к некоторому другому числу (коэффициенту сдвига).
C = (aP + K)mod M,
где a - десятичный коэффициент; K - коэффициент сдвига.
Далее, полученное число уменьшается по правилу вычитания модуля M, где M - размер алфавита. В результате имеем число (С), которое по сути является порядковым номером какой-то буквы в алфавите. Зашифрованный текст формируется из соответствующих ему алфавитных эквивалентов.
Эта формула является общей формулой моноалфавитных подстановок. При десятичном коэффициенте а = 1 получаем общую формулу шифра Цезаря.
Шифр Цезаря
Шифр Цезаря является простым примером моноалфавитной замены. Его, по имеющимся свидетельствам, использовал Юлий Цезарь. 
В данном шифре каждый знак Р из исходного алфавита замещается соответствующим числом ((Р), (((А) = 1, ((В) = 2, ?, ((Z) = 26, ((пробел) = 0(. Затем формируется эквивалентное зашифрованное число ( из уравнения
((Р) = (((Р) + 2)mod 27.
Таким образом, ((А) = ((А( ( ( = 1 ( ( =3, ((Z) = ((Z( ( ( = 26 + 2 = 28mod27 = 1 и т.д.
Значения ( затем замещаются соответствующими им буквами для получения зашифрованного текста. 
Фактически Шифр Цезаря - это любой шифр в форме 
С = (Р + К)mod М, 
где С - код символа шифртекста, Р - код символа открытого текста, K - коэффициент сдвига (К = 1(M - 1), М - размер алфавита.
Многоалфавитная замена
Методы многоалфавитной замены:
- Простая многоалфавитная замена
- Шифр Вижинера
- Шифр Тритемиуса;
Многоалфавитная замена последовательно и циклически меняет используемые алфавиты. При u-алфавитной подстановке знак m1 из исходного сообщения заменяется знаком из алфавита B1, m2 - соответствующим из алфавита B2, ?, mu - знаком из алфавита Bu, mu+1 - снова из алфавита B1 и т.д.
ВХОДНОЙ ЗНАК: m1m2m3m4m5m6m7m8m9?
АЛФАВИТ ЗАМЕНЫ: B1B2B3B4B1B2B3B4B1?
Эффект использования многоалфавитной замены в том, что она обеспечивает маскировку естественной частотной статистики исходного языка, так как конкретный знак из алфавита А может быть преобразован в несколько различных знаков шифровального алфавита В.
Степень обеспечиваемой защиты тем больше, чем длиннее период повторяемости алфавитов.
Простая многоалфавитная замена
Алфавит рассматривается в качестве кольца из М символов, где М - размер алфавита. Под каждым символом открытого текста подписываются символы ключа периодически. Размер ключа определяется количеством применяемых алфавитов. 
Символ ключа (его код) определяет сдвиг шифровального алфавита относительно исходного, для каждой буквы открытого текста.
Шифрование: Символ открытого текста сдвигают влево на число символов равное коду символа ключа.
Дешифрование: Символ шифртекста сдвигают вправо на число символов равное коду символа ключа.
Шифр Тритемиуса
В данном шифре каждый символ открытого текста и ключа рассматривается как код (порядковый номер буквы в алфавите).
Шифрование:
Под кодами символов открытого текста циклически подписываются коды символов ключа.
Символ шифртекста получается при сложении символа открытого текста с символом ключа по модулю М, где М размер алфавита.
Дешифрование:
Под кодами символов шифртекста циклически подписываются коды символов ключа.
Для получения символа открытого текста, из символа шифртекста вычитают символ ключа по модулю М, где М размер алфавита.
Шифр Вижинера
Шифр Вижинера основан на таблице Вижинера. Размер таблицы (М х М), где М размер алфавита.
Первая строка таблицы представляет собой алфавит, последующие строки сдвинуты на один символ влево по модулю М.
Шифрование:
Под символами открытого текста циклически подписываются символы ключа.
Символ открытого текста определяет столбец, а символ ключа - строку таблицы. На их пересечении находится символ шифртекста.
Дешифрование:
Под символами шифртекста циклически подписывается символ ключа.
Символ ключа определяет строку таблицы, символ шифртекста в этой строке определяет столбец. По этому столбцу в первой строке находится символ открытого текста.
G-контурная многоалфавитная подстановка
G-контурная многоалфавитная подстановка представляет собой метод шифрования, использующий g наборов алфавитов, применяемых циклически с периодом u1, u2, ..., ug.
Иначе говоря, G-контурная многоалфавитная подстановка есть g-кратное последовательное применение многоалфавитной подстановки.
Перестановка
Шифрование перестановкой заключается в том, что знаки исходного текста переставляются в соответствии с некоторым правилом.
Методы перестановки:
- Простая перестановка;
- Перестановка по маршрутам Гамильтона.
Простая перестановка
В данном методе знаки исходного текста переставляются в соответствии с ключом. Ключ представляет собой последовательность чисел (от 1 до N, где N - длина ключа), которые показывают порядок перестановки символов.
Шифрование: Исходный текст разбивается на блоки длиной N символов. Порядок символов в каждом блоке изменяется в соответствии с ключом. В результате получаем шифртекст.
Дешифрование: Шифртекст разбивается на блоки длиной N символов. Символы в блоке ставятся на места, соответствующие числам в ключе. В результате получаем открытый текст.
Перестановка по маршрутам Гамильтона
В данном методе используются так называемые маршруты Гамильтона. Каждый маршрут - это схема простой перестановки с длиной ключа равной восьми. Различают пять маршрутов Гамильтона.
Шифрование: Открытый текст разбивается на блоки по восемь символов. Символы блока переставляются по одному из пяти маршрутов. Ключ указывает порядок применения маршрутов, в отличие от простой перестановки, где он сам является схемой перестановки.
Дешифрование: Шифртекст разбивается на блоки по восемь символов. Дешифрование внутри блока происходит также как в простой перестановке. Схема перестановки определяется маршрутом Гамильтона, номер которого получают из ключа.
