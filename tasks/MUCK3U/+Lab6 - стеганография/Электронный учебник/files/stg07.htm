Обзор существующих программных продуктов.

Операционная среда Windows
Steganos for Win95 — является легкой в использовании, но все же мощной программой для шифрования файлов и скрытия их внутри BMP, DIB, VOC, WAV, ASCII, HTML — файлов. Для удобства использования программа выполнена в виде мастера. Это 32-разрядное приложение содержит собственный Shredder — программу, которая уничтожает файлы с жесткого диска. С новыми свойствами и дополнительными возможностями Steganos for Win95 является серьезным конкурентом на рынке информационной безопасности для скрытия файлов.

На рис. 20. можно наблюдать внешний вид программного продукта. В главном окне есть два основных поля с которыми в дальнейшем и ведется работа: Steganos Explorer и Secure Space. Steganos Explorer практически полностью копирует функции проводника Windows и предназначен для выбора рабочего каталога, файлы из которого будут внедряться в контейнер или куда будут сохраняться извлеченные файлы. Рассмотрим процесс внедрения файлов с тоочки зрения пользователя.:


Рис. 20.

Сначала в поле Steganos Explorer следует выбрать каталог где храняться файлы которые нужно внедрить. и нажать кнопку "Скрыть/Зашифровать" (), после чего главное окно приобретает вид, как на рис. 21.


Рис. 21.

Далее нужно выделить скрываемые файлы и перетащить их (Drag&Drop) с помощью мыши в поле Secure Space. (рис. 22.)


Рис. 22.

Перенести файлы в поле Secure Space можно также посредством кнопок "Скопировать в Secure Space" или "Перенести в Secure Space", которые находятся в поле

Далее следует убедиться, что в качестве действия над файлами выбрано "Скрыть" . Кнопки "Скрыть"(левая) и "Зашифровать"(правая) находятся в поле Secure Space.

Теперь, когда файлы выбраны, нужно выбрать контейнер, для этого надо нажать кнопку "Закрыть и защитить" в поле Secure Space. После этого на экране появится окно (см. рис.23.) предлагающее разные спопсобы выбора контейнера:

  • Выбрать посредством браузера;
  • Автоматический поиск контейнера;
  • Создание нового контейнера.

Выберем контейнер посредством браузера.


Рис. 23.

В качестве контейнера можно выбрать текстовые, звуковые и графические файлы. Если сообщение не вмещается в выбранный контейнер программа немедленно выдаст соответствующее предупреждение. Наиболее вместительными контейнерами являются 24-битные изображения. После выбора контейнера, на экране появляется окно(рис. 24.) предлагающее ввести и подтвердить пароль(стегоключ). По мере увеличения длины пароля растет надежность защиты (Password security level).


Рис. 24.

После ввода пароля и нажатия кнопки "Ok", происходит, собственно, внедрение сообщения в контейнер, и программа возвращается в исходное состояние ожидая далнейших указаний со стороны пользователя.(см. рис. 20)

Чтобы извлечь сообщение из стего нужно в поле Secure Space нажать кнопку "Извлечь/Расшифровать" () (См. рис. 20.) После нажатия этой кнопки, посредством стандартного диалога выбора файлов, следует выбрать стего. Когда стего будет выбрано, программа потребует ввода пароля(стегоключа).См. рис. 25.


Рис. 25.

Если ключ введен верно, то в поле Secure Space появляются извлеченные файлы.(рис. 26.)


Рис. 26.

Теперь осталось отметить эти файлы и перетащить с помощью мыши в выбранный зараннее каталог в поле Steganos Explorer.(рис. 27.)


Рис. 27.

Contraband — программное обеспечение, позволяющее скрывать любые файлы в 24 битовых графических файлах формата BMP. Данная программа выполнена в интерфейсе Windows 3.x, и превосходит рассмотренный выше Steganos разве что в простоте использования.

В главном окне программы(рис. 28.) сразу же имеется поле для ввода стегоключа, который будет использоваться в дальнейшем либо для сокрытия сообщения в контейнере, либо для извлечения сообщения из стего.


Рис. 28.

Чтобы скрыть сообщение, нужно ввести пароль длиной 4 символа и нажать кнопку "Combine". После этого, на экране появится окно(см. рис. 29) предлагающее выбрать файл-сообщение и файл-контейнер.


Рис. 29.

Чтобы выбрать сообщение, нужно нажать кнопку "Browse for *.*". Чтобы выбрать контейнер, нужно нажать кнопку "Browse for *.BMP". После того как сообщение и контейнер загружены, остается только нажать кнопку "START MERGE"(см. рис. 29.) По окнчании процесса внедрения, файл-стего берет себе имя файла исходного контейнера ("имяфайла.bmp"), а исходному файлу контейнера дается имя начинающееся с символа подчеркивания ("_мяфайла.bmp").

Для извлечения сообщения из стего нужно в главном окне программы ввести ключ и нажать кнопку "Extract"(см. рис. 28). После этого на экране появится окно(см. рис. 30.) предлагающее выбрать стего и ввести имя выходного файла.


Рис. 30.

Если ключ введен верно, то после выбора стего и введения имени выходного файла, происходит извлечение сообщения из стего.

Операционная среда DOS
Jsteg — программа предназначена для скрытия информации в популярном формате JPG.

FFEncode — интересная программа, которая скрывает данные в текстовом файле. Программа запускается с соответствующими параметрами из командной строки.

StegoDos — пакет программ, позволяющий выбирать изображение, скрывать в нем сообщение, отображать и сохранять изображение в другом графическом формате.

Wnstorm — пакет программ, который позволяет шифровать сообщение и скрывать его внутри графического файла PCX формата.

Операционная среда OS/2
Hide4PGP v1.1 — программа позволяет прятать информацию в файлах формата BMP, WAV и VOC, при этом для скрытия можно использовать любое число самых младших битов.

Техto — стеганографическая программа, преобразующая данные в английский текст. Текстовые файлы-контейнеры после преобразования не содержат какого-либо смысла, но достаточно близки к нормальному тексту, чтобы пройти примитивную проверку.

Wnstorm — аналогична программе для DOS. Для ПК Macintosh

Stego — позволяет внедрять данные в файлы формата PICT без изменения внешнего вида и размера PICT -файла.

Paranoid — эта программа позволяет шифровать данные по алгоритмам IDEA и DES, а затем скрывать файл в файле звукового формата.
